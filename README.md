# Matrix Arabic Rain

A terminal-based Matrix digital rain effect using Arabic script.

## Features
- Classic Matrix rain effect with Arabic characters
- Arabic letters and numbers only
- Fading trail effects
- Matrix-style glitch effects
- Variable drop speeds
- Terminal resize support

## Files
- `matrix_basic.py` - Basic version
- `matrix_numbers.py` - Adds Arabic numbers
- `matrix_fade.py` - Adds fading effects
- `matrix_glitch.py` - Adds glitch effects
- `matrix_speed.py` - Variable speeds
- `matrix_final.py` - Final optimized version
- `run.py` - Launcher script

## Usage
```bash
# Run the final version
python3 matrix_final.py

# Or use the run script
python3 run.py
